Principais formatações

Negrito / Destaque

Nessa frase, temos um termo em negrito usando a tag <b> (não semântica).

Nessa frase, temos um termo em destaque usando a tag <strong> (semântica)

Itálico / Ênfase

Nesta frase, temos um termo em itálico usando a tag <I> (não semãntica)

Nesta frase, termos um termo em enfase usando a tag <EM> (semántico)

Texto Marcado

Podemos criar também um texto marcado usando a tag <mark>

Texto grande e pequeno

Estamos crinado um texto grande e um texto pequeno neste paragrafo usando <BIG> / <SMALL>

Texto excluido / deletado

Podemos marcar um texto com excluido fazendo com que ele seja lido mais não considerado, usando <DEL>.

Texto incluido

Podemos marcar um texto como inserido para dizer que ele foi adicionado depois, usando a tag <INS>.

Texto sobrescrito

Para inserir coisas do tipo x20 + 3, usamos a tag <sup>

Texto subescrito

Para inserir coisa como H2O usamos a tag <sub>
